#  Meta Ad Performance Dashboard (Power BI)

A dynamic Power BI dashboard designed to analyze ad performance across **Facebook** and **Instagram**, helping marketers make smarter, data-driven decisions and track ROI effectively.

---

#  Project Overview

This dashboard provides a complete view of ad performance across both platforms - from impressions and clicks to conversions and ROAS.  
It helps marketing teams understand audience behavior, optimize campaigns, and maximize return on ad spend.
